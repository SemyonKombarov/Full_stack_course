Twitter Bootstrap
Свободный набор инстурментов для создания сайтов и веб-приложений

Включает в себя HTML и CSS шаблоны оформления для типографики, веб-форм, кнопок, меток, блоков навигации и прочих компонентов веб-интерфейсов, включая JS расширения. Bootstrap включает самые современные наработки в области CSS И HTML, поэтому необходимл быть внимательным при поддержке старых браузеров
